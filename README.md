# Data spreadsheet for the "CNC But Realistic" mod in Command & Conquer: Tiberian Dawn
### This spreadsheet documents all the gameplay changes the mod adds, plus the data behind them: IRL weapon ranges, the original variables I replaced, the new ones I added, and so on. Most of it is self-explanatory from the table itself. Cells with attached comments were generated by Claude Opus 4.7 to double-check the real-world context and for formatting purposes.
